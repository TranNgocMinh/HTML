## Hoàn thành các bài tập về thẻ HTML
1. Nhấn vào nút <b>Work in Repl.it</b> để đến môi trường soạn thảo<br>
2. Mở lần lượt các tập tin *.html (ví dụ title.html, heading.html,...) và hoàn thành các yêu cầu nêu trong mỗi tập tin.
3. Nhấn nút Run từ Repl.it sau khi hoàn thành mội tập tin và xem kết quả từ khung bên phải
4. Nhấn nút Version Control (hình dưới), nhập thông tin
